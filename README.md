## Sobre o repositório

Este repositório foi criado com o objetivo de armazenar e organizar as atividades complementares (ADO) desenvolvidas durante o curso.

As atividades envolvem práticas, exercícios e pequenos projetos voltados ao reforço dos conteúdos trabalhados em aula, contribuindo para o desenvolvimento técnico e acadêmico.

---

## Objetivos

* Organizar atividades acadêmicas
* Praticar conteúdos vistos em aula
* Desenvolver habilidades em desenvolvimento web
* Reforçar conceitos teóricos por meio da prática
* Manter um histórico de evolução acadêmica
